#ProjetoMinerva# 
Sistema de controle de livros.

Minerva - Minerva era a deusa romana das artes, do comércio e da sabedoria.

Tecnologias utilizadas:
Aplicação usando o desing patterns MVC.

Frontend:
	- Bootstrap
	- Javascript
	- Angular 1.5

Backend:
	- Java
	- Hibernate
	- JPA 
	- CDI
	- MySQL

Servidor: WildFly 8.2.0 Final
